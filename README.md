# TISC api

===========================

## Requirements

- NodeJS > 14.16.x
- NPM > 6..14.x

## Documentation

### Dev Enviroment

- copy `.env.example` to `.env`.
- Run `npm install`.
- Run `npm run dev`.

### Available scripts

- `dev`: Starts node in dev enviroment
- `create_db`: Create database with name in dev enviroment
- `migrate`: Migrate all tables
- `seed`: Execute all seed file
